## 2. 블록체인에대한 소개
블록체인은 무엇인가?
1. 해쉬 : 단방향, 고정된자리수 (256 비트 고정된 결과 출력 -> 16진수 문자 64자리)
2. 블록 : 버전 + 이전블록해쉬값 + 머클루트해쉬값 + 시간 + 비트 + Nonce
3. Pow : 채굴은 이상한 표현?
